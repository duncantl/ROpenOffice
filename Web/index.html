The ROpenOffice Package

Last Release: 0.4-0 (30 Aug 2011)

The ROpenOffice package provides functionality to read spreadsheets saved in the OpenOffice format. This is a Zip file containing different related XML documents. The ROpenOffice package

The primary function in the package is read.ods().

There is also an RExcel and ROOXML package for working with Microsoft Office files, and als the rexcel package by Thomas Baier.
